# Convolution
Vulkan Renderer

## Prerequisites
1. **Vulkan SDK** (Required)
2. **Visual Studio 2022** (C++ Desktop)
3. **CMake** & **Git**

## Build
1. Generate project: `cmake -S . -B build`
2. Build: `cmake --build build --config Debug`

*All other dependencies (GLFW, ImGui, etc.) should be downloaded automatically through git.*
